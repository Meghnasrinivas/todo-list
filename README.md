Todo App :

This is a simple Todo App built using HTML, CSS, and JavaScript. It allows users to add tasks along with a due date and manage them easily through a clean interface.

Features :

Add a todo item with a due date

Display all added todo items

Delete individual todo items

Instant updates without page reload

Technologies Used :

HTML

CSS

JavaScript

How It Works :

The user enters a task and selects a date

Clicking the Add button stores the task in a list

Tasks are dynamically displayed on the screen

Each task has a Delete button to remove it

How to Run :

Open the HTML file in any web browser

Enter a todo item and select a date

Click Add to see the task appear

Purpose :

This project was created to practice JavaScript fundamentals, DOM manipulation, and basic frontend development concepts.
